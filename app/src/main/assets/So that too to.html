so ... that ➫ too ... to


  • so ... that သည် ရှေ့ဝါကျမှ နာမဝိသေသန၏ အကျိုးဆက်ကို ဖော်ပြသော ပုံစံဖြစ်သည်
  • အေးလွန်းလို့ အပြင်မထွက်နိုင်, ပျင်းလွန်းတဲ့အတွက် စာမေးပွဲမအောင်ခဲ့ စသည့် ပုံစံများဖြစ်သည်
  • so ... that မှ too ... to သို့ ပြောင်းရာတွင် so နေရာတွင် too ရေးပါ
  • that ဖြုတ်ပါ
  • Subject များတူလျင် that ဝါကျမှ Subject ကိုဖြုတ်ပါ
  • Subject များမတူလျင် that ဝါကျမှ Subject ကို for ခံပြီး ရေးပေးရသည်
    • I ➟ for me
    he ➟ fo him
    she ➟ for her
    we ➟ for us
    they ➟ for them
  • can/ could + not ကိုဖြုတ်ပြီး to နှင့်ဆက်ပါ
  • Subject မတူသော ဝါကျများကို ဆက်သွယ်ရာတွင် Verb နောက်ရှိ ရှေ့ဝါကျမှ Subject ကို ညွှန်းသော စကားလုံးများ ဖြုတ်ပါ
Differences between so and too

so သည် အများအားဖြင့် very နှင့် အဓိပ္ပါယ်တူသည်

◾I'm so glad to see you.
◾We have so much to do.
◾I wish she didn't drive so fast.

so ကို that-clause ရှေ့တွင် ထားသုံးနိုင်သော်လည်း very ကို ထိုကဲ့သို့ အသုံးမပြုနိုင်ပါ

◾It was so cold that we stopped playing.
(NOT It was very cold that we stopped playing. )
◾He spoke so fast that nobody could understand.
(NOT He spoke very fast that ...)

too သည် လိုတာထက်ပိုနေသော အခြေအနေကိုပြပြီး အငြင်း အဓိပ္ပါယ် သက်ရောက်သည်

◾It was very cold, but we went out.
◾It was too cold to go out, so we stayed at home.

ပထမဝါကျတွင် very သည် အအေးဓာတ် များပြားကြောင်းကို ပြောသည်... အလွန်အေးသည် သို့သော် ကျွန်တော်တို့ အပြင်ထွက်ခဲ့ကြသည်... ဒုတိယဝါကျတွင် too သည် လိုတာထက် ပိုအေးကြောင်းကို ပြသည်... အေးလွန်းလို့ အပြင်မထွက်နိုင် ဟုဆိုလိုသည်

အောက်ပါဝါကျနှစ်ကြောင်းကို နှိုင်းယှဉ်ကြည့်ပါ
◾He is a very intelligent child.
◾He's too intelligent for his class - he's not learning anything.



Examples:
1. Ma Ma is so fat that she cannot run very fast.
Ma Ma is too fat to run very fast.
2. Tun Tun was so tired that he could not study any lessons that night.
Tun Tun was too tired to study any lessons that night.
3. The book is so expensive that he cannot buy it.
The book is too expensive for him to buy.
4. The question was so difficult that I could not answer it.
The question was too difficult for me to answer.
