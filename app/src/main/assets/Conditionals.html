Conditionals


Introduction

အခြေအနေတစ်ခု ဖြစ်ပျက်လျင် နောက်အခြေအနေတစ်ခု ဖြစ်ပျက်မည်ကို ပြောလိုသည့်အခါ ဝါကျနှစ်ကြောင်းကို if ဖြင့်ဆက်ပြီး ရေးနိုင်သည်... if အသုံးပြုပြီး ရေးထားသော ဝါကျတည်ဆောက်ပုံကို conditional structure ဟုခေါ်သည်

◾If you get here before eight, we can catch the early train.
◾Oil floats if you pour it on water.

Conditional: Types

conditional အမျိုးအစား သုံးမျိုးရှိသည်

  • Type One: if + present .... will + infinitive
    ◾If we play tennis, I will win.

  • Type Two: if + past .... would + infinitive
    ◾If we played tennis, I would win.

  • Type Three: if + past perfect .... would have + past participle
    ◾If we had played tennis, I would have won.
conditional type one သည် ဖြစ်နိုင်ခြေရှိသော အခြေအနေမျိုးကို ဖော်ပြသည်... ဝါကျ တည်ဆောက်ပုံမှာ if-clause တွင် present (V1) ဖြစ်ပြီး main clause တွင် simple future (will + V1) ဖြစ်သည်

◾If it rains, we will not go out.
◾If Nilar is late, her boss will be mad.
◾If you don't hurry, you will be late.

conditional type two သည် ယခုလက်ရှိတွင် ဖြစ်မနေသော အခြေအနေ (သို့) စိတ်ကူးယဉ် အခြေအနေကို ဖော်ပြသည်... ငါသာ ငှက်ကလေးတစ်ကောင်ဖြစ်ရင်..., ထီပေါက်ခဲ့ရင်... စသော အခြေအနေများအတွက် အသုံးပြုသည်... ဝါကျတည်ဆောက်ပုံမှာ if-clause တွင် past(V2) ဖြစ်ပြီး main clause တွင် would + infinitive(V1) ဖြစ်သည်

◾If I were a bird, I would fly away.
◾If I knew her name, I would tell you.
◾If I had a big house, I would be very happy.
◾What would you do if you lost your job ?

Conditionals: Tenses

ပုံမှန်အားဖြင့် မှန်ကန်နေသည့် အကြောင်းအရာများကို ပြောဆိုသည့်အခါ main clause တွင် future အစား present/ past ကိုသာ အသုံးပြုသည်

◾Oil floats if you pour it on water. (NOT ... Oil will float )
◾If you heat ice, it melts.
◾If Ko Ko didn't come to work yesterday, he was probably ill.

ယခုလို အသုံးပြုပုံကို အချို့က conditional type zero ဟု ခေါ်ကြသည်


conditional type three ကို unreal past ဟုလည်း ခေါ်သည်... အတိတ်ကာလက ဖြစ်ပျက်ခဲ့သော/ မဖြစ်ပျက်ခဲ့သော အကြောင်းအရာနှင့် ၎င်း၏ အကျိုးဆက်ကို ဖော်ပြသည်... ဝါကျ တည်ဆောက်ပုံမှာ if-clause တွင် past perfect (had + v3) ဖြစ်ပြီး main clause တွင် would have + past participle(V3) ပုံစံ ဖြစ်သည်

◾If you had tried hard, you would have passed the exam.
(အခုတော့ မကြိုးစားခဲ့သည့် အကျိုးဆက်ကြောင့် စာမေးပွဲ မအောင်ခဲ့ ဟုဆိုလိုသည်)
◾If you had asked me, I would have told you.
◾I would have been in bad trouble if he had not helped me.
သူ သာမကူညီခဲ့ရင် ဒုက္ခ ကောင်းကောင်းရောက်မှာ
(အခုတော့ သူ ကူညီခဲ့လို့ ဒုက္ခမရောက်ဘူး...)


Conditionals in action

Case one:

I don't have a car, so I'm not able to go wherever I want.

ယခု ဝါကျကိုကြည့်လျင် ကားသာရှိလျင် သွားချင်ရာ သွားနိုင်မည် ဟု စိတ်ကူးယဉ်ပုံစံဖြစ်ကြောင်း တွေ့ရမည်... conditional type two ကို အသုံးပြုရမည်... အထက်ပါ ဝါကျကို အောက်ပါအတိုင်း ရေးသားနိုင်သည်

◾If I had a car, I would be able to go wherever I want.

Case two:

My father didn't earn much money, so life wasn't easy for us.

ယခုဝါကျသည် အတိတ်ကမဖြစ်ပျက်ခဲ့သည့် အကြောင်းအရာကို ဖော်ပြသည်... အဖေသာ ပိုက်ဆံများများဝင်ခဲ့ရင် ဘဝက လွယ်ကူချောမွေ့ခဲ့မယ် ဟု ပြန်ပြင်မရသည့် အတိတ်အကြာင်း ပြောဆိုနေသောကြောင့် conditional type three သုံးပြီး ယခုလို ရေးနိုင်သည်

◾If father had earned a lot of money, life would have been easy for us.



Mixed conditionals : another headache?

တစ်ခါတစ်ရံ conditional များကို type two နှင့် type three အတူရောပြီး အသုံးပြုတတ်သည်

◾If I had worked harder at school, I would have a better job now.
(ကျောင်းတုန်းက စာမကြိုးစားခဲ့တဲ့ ရလဒ်က အခု အလုပ်ကောင်းကောင်းမရဘူး လို့ဆိုလိုသည်)

if-clause တွင် conditional type three အသုံးပြုပြီး main clause တွင် conditional type two အသုံးပြုထားသည်

One more example:
◾If we had looked at the map, we wouldn't be lost.

◾If you weren't afraid of spiders, you would have picked it up and put it outside.
( ပင့်ကူကို ကြောက်တတ်လို့ ကောက်မကိုင်ခဲ့တာ... )

if-clause တွင် conditional type two အသုံးပြုပြီး main clause တွင် conditional type three အသုံးပြုထားသည်
